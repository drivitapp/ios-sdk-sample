DrivitTripEditInterface

DrivitTripEditInterface is a protocol that enables the edition of a trip.

  • getOrigin()

    The Address were the trip started, this address is an operation that can fail.

  • getDestination()

    The Address were the trip ended, this address is an operation that can fail.

  • getDistance()

    The distance calculated by all the locations that were registered for the trip.

  • isSoftDelete()

    The consume is calculated depending on the distance and the vehicle consume.

  • getConsumption()

    The consume is calculated depending on the distance and the vehicle consume.

  • getFuelPrice()

    The fuel price.

  • setSoftDelete()

    Set the trip as soft deleted.

  • setOrigin(_:)

    Setup the Address were the trip started.

  • setDestination(_:)

    Setup the Address were the trip ended.

  • setDistance(_:)

    Setup the Total Distance.

  • setaFuelConsumption(_:)

    Setup the Total Consumption.

  • setFuelInfo(withConsumption:withFuelPrice:withDistance:)

    Setup the Fuel Info.
