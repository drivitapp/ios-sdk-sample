  • fuelConsumption(guid:)

    Returns the fuel consumption of the trip, in l/100km. This method may return one of three cases:

    • If the user manually edited the consumption of the trip, it will return the value inserted
    • If a good-accuracy fuel consumption estimate exists, it will return it
    • If the consumption analysis wasn’t performed yet or if it’s low-accuracy estimate, it will fallback to the vehicle average

    Declaration

    Swift

    func rejectedReason(guid: String, mergeWithServer: Bool) -> Int

    Parameters

    guid

    Trip identifier
    mergeWithServer

    False to retrieve a value without analysis from Drivit server (e.g. the user may reject the trip directly on the phone), true if you want to merge both results. When merging, any potential user input will always have precedence over the analysis from the Drivit server.
