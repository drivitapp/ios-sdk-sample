DrivitLocationInterface

DrivitLocationInterface is the main representation of a Location object.

  • getLocation()

    The location coordinates were this location was registered.

  • getTimestamp()

    The date were this location was registered.

  • getAccuracy()

    The location accuracy for this location.

  • getSpeed()

    The speed registered.

  • getDirection()

    The direction registered.

  • getAltitude()

    The altitude registered.

  • getCLLocation()

    The location as a CLLocation object.
